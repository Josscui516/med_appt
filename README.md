# Medical Appointment Booking Website

## Project Overview
The Medical Appointment Booking Website is designed to help patients easily connect with doctors and schedule medical consultations online. This platform is part of the StayHealthy digital initiative to improve healthcare accessibility in remote areas.

## Features
- Online appointment booking
- Doctor listing with ratings and reviews
- Consultation feedback system
- Patient profile management
- Easy website navigation

## Technologies Used
- HTML
- CSS
- ReactJS
- Git and GitHub

## Purpose
This project is part of the Front-End Development Capstone Project. The goal is to design and develop a user-friendly platform that enables patients to schedule appointments and interact with doctors efficiently.
